# Юридические документы

Сюда кладутся PDF-файлы, на которые ссылается футер и чекбоксы в форме обратной связи.

Ожидаемые имена файлов (не переименовывать — на них завязаны ссылки в коде):

- `privacy-policy.pdf` — Политика конфиденциальности
- `personal-data-policy.pdf` — Политика обработки персональных данных

Чтобы заменить документ — просто положи новый PDF с тем же именем поверх старого.
Файлы из `public/` отдаются по прямому пути, например: `https://koporych.ru/docs/privacy-policy.pdf`
